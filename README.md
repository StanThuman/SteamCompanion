# Steam Companion
Web application that will use the Steam API to provide tools for Steam users such as comparing games with multiple friends. May add more tools later.

### Technologies

* Front-End
    * Reactjs
    * Reduxjs
    * Lodash
    * Material-UI
    * Jss
    
* Back-End
    * Nodejs
    * Express
    * Server-Side Rendering
    
* Unit-Tests
    * Enzyme
    * Mocha
    * Chai
